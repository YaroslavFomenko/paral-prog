# Лабораторная работа №3
## Параллельное умножение матриц с использованием MPI

В данной работе алгоритм умножения матриц модифицирован для параллельной работы с использованием технологии MPI. Каждый процесс вычисляет свою часть строк результирующей матрицы.

Используемые размеры матриц: 200, 400, 800, 1200, 1600, 2000.
Количество процессов: 1, 2, 4, 8.

---

## Состав проекта

| Файл | Описание |
|------|----------|
| gen.py | Генерация исходных матриц |
| main_mpi.cpp | Параллельная версия умножения с MPI |
| plot_mpi.py | Построение графика |

---

## Порядок выполнения

### 1. Генерация матриц

python3 gen.py

### 2. Компиляция

mpic++ -std=c++11 -O2 -o main_mpi main_mpi.cpp

### 3. Запуск

mpirun -n 1 ./main_mpi
mpirun -n 2 ./main_mpi
mpirun -n 4 ./main_mpi
mpirun -n 8 ./main_mpi

### 4. График

python3 plot_mpi.py

## Вывод

1. Реализовано умножение матриц с использованием MPI.
2. При увеличении числа процессов время выполнения существенно уменьшается.
3. Максимальное ускорение достигнуто на 8 процессах для размера 2000: с 3.945 до 0.751 сек (в 5.25 раз).
4. На малых размерах накладные расходы на коммуникацию снижают эффективность.
5. MPI эффективно распределяет вычисления между процессами на разных ядрах.
